## 🎯 **核心原则提醒**

### **后端职责边界**
```
✅ 后端关注: 接口合理性、功能完备性、数据安全、性能优化
❌ 前端关注: 移动端适配、UI交互、浏览器兼容性
```

### **强制性规范**
1. **路由设计**: 子路由不设prefix，主路由统一添加
2. **权限控制**: 所有业务接口必须添加权限验证
3. **参数验证**: 使用Pydantic严格验证输入
4. **错误处理**: 统一错误响应格式
5. **代码质量**: 完整注释、测试覆盖
